Accessing an array

An array is like a line of items. Each item has an index. An array of people's names might look like this: 

  let array1 = ["andy", "betty", "cathy", "danna", "edward"];

We can access the first person in the line with the index for that person. (The index starts at 0, because we are talking to a computer!) 

  array1 = ["andy", "betty", "cathy", "danna", "edward"];
  document.write(array1[0]);

Assigning to an array

You can also change an array. An array is mutable. That means "changable". You mutate an array by using the = assignment operator, just as you would for a variable.

Let's swap people in line.

We start with this... 

  array1 = ["andy", "betty", "cathy", "danna", "edward"];

We want to end with this... 

  ["betty", "andy", "cathy", "danna", "edward"];

We can access them by index and assign the new names. 

  array1[0] = "betty";
  array1[1] = "andy";
  
  // array1 will be this
  // ["betty", "andy", "cathy", "danna", "edward"];
